# Nafis Jaya Accu

Website profesional untuk layanan aki mobil, aki motor, setrum aki, tukar tambah, dan perlengkapan aki di area Surabaya, Sidoarjo, dan sekitarnya.

## Fitur utama

- Sidebar glass control panel dengan navigasi section.
- Hero Electric Glass Service Console.
- Search katalog aki berdasarkan kode aki, kendaraan, kapasitas, dan kategori.
- Ringkasan pesanan dengan draft tersimpan di `localStorage`.
- Generator pesan WhatsApp otomatis berisi aki pilihan, kendaraan, kebutuhan, dan lokasi.
- Section layanan, cara pesan, area layanan, FAQ, dan CTA akhir.
- Leaflet map dengan marker toko.
- Tombol Google Maps menuju `https://maps.app.goo.gl/E94fTsWyBTr7psELA`.
- Creator credit menuju Instagram `@nfs_ky07`.
- SEO metadata, Open Graph image, robots, sitemap, dan structured data LocalBusiness/AutoPartsStore.

## Menjalankan project

```bash
npm install
npm run dev
```

Buka `http://localhost:3000`.

## Build production

```bash
npm run lint
npm run typecheck
npm run build
```

## Catatan desain

Desain memakai pendekatan golden ratio secara proporsional melalui layout 61.8/38.2, spacing scale, section rhythm, card radius, dan komposisi panel. Tema visual diarahkan sebagai **Electric Glass Service Console**: dark graphite, electric cyan, battery amber, glassmorphism, dan UI modern yang tetap formal serta jelas dibaca.

## Catatan data

Katalog aki adalah panduan awal. Kecocokan kendaraan, ukuran, terminal, posisi kutub, stok, harga, merek, dan opsi tukar tambah tetap perlu dikonfirmasi via WhatsApp.
